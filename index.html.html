📝 Существительные (-tion, -ment)
Для образования существительных от глаголов часто используются суффиксы -tion и -ment.
create → creation
develop → development
argue → argument
📝 Прилагательные (-ful, -less, -able)
Суффиксы -ful (полный), -less (без), -able (способный).
care → careful / careless
comfort → comfortable
help → helpful / helpless
📝 Наречия (-ly)
Для образования наречий от прилагательных добавляется суффикс -ly.
quick → quickly
careful → carefully
happy → happily
📝 Отрицание (un-, in-, im-)
Приставки un-, in-, im- дают отрицательное значение.
happy → unhappy
possible → impossible
correct → incorrect
📝 Существительные (-ness, -ity)
Суффиксы -ness и -ity образуют абстрактные существительные.
happy → happiness
active → activity
kind → kindness
📝 Прилагательные (-ous, -ive)
Суффиксы -ous и -ive образуют прилагательные.
danger → dangerous
create → creative
success → successful
